# ElectroMart - E-Commerce Platform

ElectroMart hocche akta modern, full-stack e-commerce web application jekhane electronics products pawa zabe. Ei project-ti purnaangobhabe **TypeScript** ebong **Next.js (App Router)** use kore toiri kora hoyeche jate frontend highly optimized, SEO-friendly ebong type-safe thake.

## 🚀 Live Demo

Apni project-ti nicher link-e live dekhte parben:
👉 **[ElectroMart Live Website](https://electro-mart-sigma.vercel.app)**

---

## 🔥 Key Features

Ei project-e nicher dynamic e-commerce features gulo include kora hoyeche:

- **Product Showcase & Details:** Shob electronics products dynamic grid layout-e display kora hoy ebong protiti product-er specific detail page (price, description, specification) royeche.
- **Advanced Search & Filtering:** Visual optimization-er shathe user-ra khub shojei nirdishto product search korte parbe ebong price range, brand ba ratings diye filter korte parbe.
- **Category-wise Browsing:** Category dynamic routing system (e.g., Laptops, Phones, Accessories) er maddhome alada alada dynamic category show kora hoy.
- **Interactive Shopping Cart:** Real-time dynamic cart system, jekhane user product quantity barhate/komahte ebong dynamic price updates shoho cart list modify korte parbe.
- **Responsive UI/UX:** Full responsive grid dashboard design jeta Mobile, Tablet ebong Desktop device-er jonno optimized.

---

## 🛠️ Tech Stack & Architecture

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript (Type-safe robust coding)
- **Styling & UI:** Tailwind CSS (Modern, clean and responsive utility layout)
- **Fonts Optimization:** [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load **Geist** font family.
